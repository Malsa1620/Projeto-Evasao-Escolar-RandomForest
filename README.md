# Projeto de Machine Learning - Previsão de Evasão Escolar

## Objetivo

Este projeto utiliza o algoritmo Random Forest para prever o risco de evasão escolar com base em dados de alunos.

## Tecnologias Utilizadas

* Python
* Pandas
* Scikit-Learn
* Matplotlib

## Variáveis Utilizadas

* Faltas
* Média
* Participação

## Como Executar

### Instalar bibliotecas

```bash
pip install pandas
pip install scikit-learn
pip install matplotlib
```

### Executar o modelo

```bash
python main.py
```

### Gerar gráfico

```bash
python grafico.py
```

## Funcionamento

O algoritmo Random Forest analisa exemplos anteriores de alunos e identifica padrões relacionados à evasão escolar.

## Aplicação Prática

A solução pode auxiliar escolas e universidades a identificar alunos em risco e agir preventivamente para reduzir a evasão.
